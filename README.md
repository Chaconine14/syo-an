# 架空のコンドミニアムLP

## 概要
架空のコンドミニアム「松庵」をテーマにしたランディングページです。
視覚的な世界観と情報の整理を両立させ、観光で訪れるイメージが湧くような構成を心がけました。
背景画像や装飾など、落ち着いた和モダンの世界観に重きを置きました。

## コメント
このLPは、実務を想定した作品を作りたいという思いから制作しました。
ターゲットやコンセプトから考案し、デザインまで落とし込みをしています。

インバウンドを想定し、日本語だけでなく英語にも対応した設計です。
CTAのホバーアニメーション、Swiperの導入、メインビジュアルのズームインアニメーションなど、細部の動きにもこだわりました。

このLP制作を通して、これまで積み重ねてきた学びを一つの形にできたと感じています。
