# ポートフォリオサイト

## 概要
自分自身のスキルや制作実績をまとめたポートフォリオサイトです。  
デザインからコーディングまで一貫して行い、「自分らしさ」と「技術力」が伝わる構成を意識しました。

## コメント
このポートフォリオサイトは、初めてデザインから自分の手で制作した作品です。  
自分のスキルや制作実績をわかりやすく伝えることを第一にしつつ、Web制作への想いを込めたロゴも自作し、見る人がワクワクするようなサイトを目指しました。

FLOCSS設計を用い、メディアクエリや変数をmixinで管理するなど、構造の整ったレスポンシブ対応を意識。  
また、スクール課題では扱わなかった **GSAPを独学で導入** し、動きを通して自分の世界観を表現しています。

この作品を通して、“模写から自作へ”と一歩進んだ経験を得ました。  
学んだ技術を自分なりに組み合わせ、形にすることで「表現したい世界観を自分の力で実装する楽しさ」を感じられた作品です。
